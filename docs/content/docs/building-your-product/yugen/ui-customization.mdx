The UI for this template is based on **shadcn/ui**, which means you can customize things however you want.

## TweakCN

A tool called [TweakCN](https://tweakcn.com) has:
- Pre-built themes you can use
- Custom theme builder

### Apply a Theme

**Option 1:** Copy and paste the CSS into your project

**Option 2:** Run the TweakCN command and it's done for you

```bash
npx tweakcn apply theme-name
```

The template theme updates seamlessly.
